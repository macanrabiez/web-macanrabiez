Ini Adalah Sebuah Halaman Tutorial, Semoga Dapat Membantu

Hai buat kamu pengunjung situs Macan Rabiez yang sedang mencari tutorial HTML dan CSS dasar mungkin laman ini bisa jadi solusinya,ya mungkin tutorial yang akan saya sampaikan hanya sedikit, namun saya harap ini bisa bermanfaat bagi kalian yang masih pemula dalam dunia Web HTML dan CSS sama seperti saya.


Tutorial kali ini yaitu bagaimana cara menampilkan icon sosial media tanpa menggunakan berkas gambar melainkan dengan tag HTML dan CSS Font Awesome 4 namun sebelum menggunakannya ada yang harus kalian lakukan terlebih dahulu yaitu mengcopy script dibawah ini.



Kemudian paste script tersebut dibagian head HTML blog atau website kalian seperti pada gambar dibawah ini, karena di script tersebut terdapat link yang berfungsi sebagai link pemanggil CSS ke halaman website atau blog kita.
